Python中对ADD字节码指令的实现中，判断溢出代码如下：
```c++
sum = x + y;
if ((sum ^ x ) < 0 && (sum ^ y) < 0)
    overflow
```
溢出时，sum与(x和y)的符号不一样。上面if判断cover了正溢出和负溢出。
